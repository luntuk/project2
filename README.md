# Прогноз опадів — Streamlit додаток
Простий веб-додаток на **Streamlit**, який дозволяє:
- завантажувати історичні щоденні метеодані з сервісу **Open-Meteo**,
- навчати модель класифікації (Random Forest) на основі цих даних,
- прогнозувати наявність опадів (так/ні) на наступний день або для обраної дати з історичних даних,
- показувати ймовірність опадів та основні метрики якості моделі.
## Основні можливості
- Отримання даних за координатами та періодом (latitude, longitude, дати початку та кінця)
- Збереження даних у файл `weather_daily.csv`
- Навчання моделі RandomForestClassifier з використанням лагових ознак (попередній день)
- Оцінка моделі: Accuracy, Classification Report
- Прогноз опадів з виводом ймовірності
- Інтерфейс з кнопками та можливістю завантаження готового CSV
